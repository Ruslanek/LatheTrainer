# LatheTrainer

Symulator tokarki dla nauki obsługi obrabiarek skrawających.  
Projekt edukacyjny: tryb **manualny** oraz podstawowy tryb **CNC**.

## Funkcje (planowane)

### Etap 1 – Manual Lathe

- Wybór półfabrykatu (średnica, długość, materiał: stal, aluminium, mosiądz).
- Różne narzędzia tokarskie (noże zewnętrzne, przecinak itd.).
- Ruch narzędzia w osiach X/Z z ograniczeniami.
- Symulacja zdejmowania materiału (profil wałka).
- Efekt odcięcia detalu i jego „upadku”.
- Zapis prostego rysunku gotowego detalu (PNG).

### Etap 2 – CNC Lathe (tryb treningowy)

- Panel sterowania CNC (start/stop, wybór programu).
- Proste programy testowe (pseudokod G-code).
- Podgląd ruchu narzędzia według programu.

## Technologie

- Unity (2D)
- C#

## Cel projektu

Stworzenie prostego, ale rozwojowego trenażera tokarki, który można wykorzystać
do nauki podstaw obróbki oraz jako projekt portfolio.
